Over the course of three, or so, weeks, we were tasked to create a "lite" version of Maya.

Week 1: Half-Edge Mesh 10/26/16

Half-Edge Mesh Class


Trying to figure out how to implement the Half-Edge Mesh data structure was tricky. On the assignment specifications, Adam suggested to make a physical paper cube to see how everything would be linked together, which was extremely helpful.

We had to draw a cube based on the data structure implementation. I began by hardcoding a cube by its vertices. This basically was given to us from the previous scene graph assignment. I think this was where my downfall was. I was hard coding the half-edges, syms, vertices and, faces to make the data structure for the cube. This took an unnecessary amount of time.

I should have started with the OBJ importing so I can generalize the data structure.

GUI


For this week, we didn't have to make an extensive GUI. We just had to make sure the vertices, faces, and half-edges that were created are shown on the GUI. When a vertex, face, or half-edge is selected, their respective counterpart on the mesh should be highlighted.

One tip that we were given for this part was to use signals and slots from the GUI to MyGL to draw the selected vertex/face/half-edge in a different color. I ran into a lot of trouble with this unfortunately. When I promoted the Qt widget, it disappeared from the screen. Even the TA's couldn't figure out why this was happening.

To combat this, I simply called a function whenever a Qt list widget item was selected. The function would get the half-edge mesh data from MyGL to MainWindow where the GUI functions are. I traversed the list of faces/vertices/halfedges in MainWindow and compared it to the typecasted Qt list widget item. Once I found what I was looking for, I returned the selected item to MyGL so its respective part can be redrawn in a different color.

It's not the most elagant approach, but it worked.

Another odd problem I had here was that some of the edges that made up a face would not be drawn. A selected halfedge would be drawn and a selected vertex would be drawn, but not a full face. For the cube, 2/4 halfedges would have been recolored on the face. I figured this was a minor problem and decided to move on to harder portions of this assignment.
