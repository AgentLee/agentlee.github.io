GPU Path Tracer CUDA

This is a basic Path Tracer built in CUDA. I provide an in-depth performance analysis as well as other algorithm descriptions on my Github Repo.
Many of the features in this project are based on my CPU Path Tracer. However, I was able to explore subsurface scattering for this project (purple cube in the above render).
