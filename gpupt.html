GPU Path Tracer CUDA

This is a very simple Path Tracer built in CUDA.

Features:

  • Materials
    • Diffuse
    • Specular
    • Transmissive
    • Subsurface Scattering
  • Shading/Integration
    • Naive BSDF Sampling
    • Direct Lighting
  • Anti-Aliasing
  • Depth of Field

I provide an in-depth performance analysis as well as overview on some of the algorithms I used for this project on my repo.

I hope to expand upon this project in the future with MIS, BVH or KD-tree acceleration, among other features.
