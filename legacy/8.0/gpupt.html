GPU Path Tracer CUDA

This is a basic Path Tracer built in CUDA. I provide an in-depth performance analysis as well as other algorithm descriptions on my Github Repo.
Many of the features in this project are based on my CPU Path Tracer. However, I was able to explore subsurface scattering for this project (purple cube in the above render).

Depth of Field

Anti-Aliasing

Depth of Field is based on the PBRT implementation. Anti-Aliasing was achieved using a random jitter of the pixel.

Materials

Transmissive

For transmissive objects, I used Schlick's Approximation for the Fresnel term.

Subsurface Scattering

I did not have time to read through and implement Jensen's subsurface scattering paper. However, I was still able to achieve a subsurface scattering effect using a random walk through the surface. I go much more in depth on this algorithm in my readme. Even on the GPU, the convergence rate for materials with subsurface scattering, is pretty slow.
