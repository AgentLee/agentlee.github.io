RU Studying

Web App [Python/Flask] Summer 2015

Personal Project

During operating hours, it's pretty difficult to find a classroom to study in. There always seems to be a lecture going on when you want to study in a room. RU Studying solves this problem.

RU Studying parses the data obtained from the schedule of classes from the Rutgers API. Students choose the campus they are in (1), then students pick the building on campus (2), and finally students can view a list of classrooms that are available (based off the current time) for studying (3).

JLinux

Simple Shell [C] (Spring 2015)

Course: Operating Systems (CS416)
Teammates: Sehaj Singh and Nikki Nagar

The goal of this assignment was to create a simple shell that can carry out various unix commands. We were given a text file with every word from Moby Dick and we had to perform functions on the text file.

Implemented various functions such as:

  • ls
    • lists everything in the directory
  • ws
    • counts lines, words, and characters
  • cat
    • prints contents of a file in the terminal
  • tr
    • translates letters from uppercase to lowercase and vice versa
  • sed
    • runs sed to delete
  • sort
    • sorts output
  • uniq
    • combines adjacent lines

Example Commands:

  • ls | wc -l
    • Number of files and directories in current directory.
  • cat moby.txt
    • Outputs moby.txt into terminal.
  • cat moby.txt |tr A-Z a-z|tr -C a-z '\n' |sed '/^$/d' |sort|uniq -c|sort -nr|sed 10q
    • Finds top ten words in moby.txt and count occurance.

Special Permission Number Registration System

Web App [Java/JSP, MySQL] Summer 2014

Course: Principles of Data and Knowledge Management (CS336)
Teammates: Allison Welsh, Joe Pestonit, Yuan Tian

SPN's are given to students who want to enroll in a course but the course has reached its its registration cap. This application was to allow students to login with their RUID and password, find the course they want, and apply for the SPN. Professors can also login with their RUID and password, check the waiting list for SPN's, check each student's grades/transcripts, and determine if the student receives it or not. Admins have access to all aspects of the system.

Our system used Java for the backend and MySQL for the database. We developed the schema, database, and website for the entire application.
