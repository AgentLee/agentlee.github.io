Path Tracer

C++ and Qt
Description

Path Tracing is a rendering technique that renders 3D scenes as realistically as possible. Monte Carlo estimation is used to approximate the light energy from all directions for any point in a scene.
Features global illumination, multiple importance sampling, and volumetric rendering.

Heatstroke

WebGL and JavaScript
Github Repo | Description | Live Demo

Heatstroke is a real-time physically-based volumetric renderer that is based on EA's Frostbite Engine presented at Siggraph 2014 by Sebastien Hillaire. Volumetric rendering is a technique to create fog, smoke, and other similar effects.
Features heterogeneous and homogenous media, god rays, shadow mapping, and tone mapping.
