Hand of God Unreal Engine


Github Repo
Teammates: Michael Stauffer and Aman Sachan

Hand of God is a cooperative VR endless runner. One player takes control of the runner and the other player takes on the power of God.
The runner must collect different resources to fend off enemies and also for the God.
The God player uses the collected resources to create new platforms that connect the gaps between other platforms. The God can also use the platform to catch the runner if they fall and block enemies from reaching the runner.

Hand of God Unreal Engine

This is a basic Path Tracer built in CUDA. I provide an in-depth performance analysis as well as other algorithm descriptions on my Github Repo.
Many of the features in this project are based on my CPU Path Tracer. However, I was able to explore subsurface scattering for this project (purple cube in the above render).
