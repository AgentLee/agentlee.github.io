Mesh+Rig Editor C++, OpenGL, Qt

OBJ files are parsed, loaded into a half-edge mesh data structure, and are drawn onto the screen with OpenGL. JSON files contain skeleton data which are parsed and drawn onto the screen with OpenGL. When the user presses "Skin Rig", the skeleton gets skinned to the mesh. This means that when the user manipulates a joint on the skeleton, the parts of the mesh that are bound to the joint move as well. I also implemented a basic heat map to allow the user to adjust the influence each joint has on the mesh.
